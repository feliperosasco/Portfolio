Acerca de mi

Hola, soy Felipe Rosasco, un programador apasionado por crear soluciones digitales. Actualmente estudio la Tecnicatura Universitaria de Programacion en la UTN.

Mi camino como programador comenzó hace cuatro años, demostrando mi iniciativa y mi habilidad para el aprendizaje autodidacta. Esta experiencia me enseñó que la paciencia, la creatividad y la perseverancia son las claves para abordar cualquier desafío.

Actualmente, estoy aplicando estos principios en sistemas de gestion y busco oportunidades donde pueda seguir creciendo y aportando mis conocimientos para desarrollar soluciones innovadoras.

Proyectos

    Sistema de Gestión de Librería

  • Descripción: Un sistema de gestión de consola para una librería, que permite administrar el inventario de libros, el registro de préstamos y la información de los usuarios.
  • Tecnologías: C++, Programación Orientada a Objetos (POO).
  • Aprendizaje y Rol:En este proyecto apliqué los principios de la POO para crear clases y objetos que representan los componentes de la librería, como libros y usuarios, lo que me ayudó a crear un código modular y fácil de mantener.

    Aplicación para gestión de Artículos

  • Descripción:Aplicación que permite gestionar el inventario a través de funcionalidades clave como el listado de artículos, búsqueda avanzada por criterios, y una interfaz completa para agregar, modificar y eliminar ítems.
  • Tecnologías:C#, POO, Winforms.
  • Aprendizaje y rol:Apliqué a profundidad los principios de la Programación Orientada a Objetos para crear un código modular y escalable. Utilicé WinForms para desarrollar una interfaz de usuario intuitiva y robusta, lo que me permitió combinar la lógica de negocio con una experiencia de usuario fluida.
